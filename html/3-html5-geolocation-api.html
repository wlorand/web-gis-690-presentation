HTML5 Geolocation API & How JavaScript Swallowed the Web

"The dominance of the web platform means the rise of the API; The rise of the API means the dominance of JavaScript as it is built into every web browser, both desktop and mobile." (Detwiler, 2016)
"Atwood's Law: Any application that can be written in JavaScript, will eventually be written in JavaScript." (Atwood, 2007)

Technology at a Glance

  • The HTML5 Geolocation API makes web pages location aware, and provides JavaScript methods for acquiring and using location data in web pages
    This is how all location-aware apps on your mobile devices know your location (if you allow it)
  • The HTML5 geolocation object provides a getCurrentPosition() method that gives you back lat/long and more
  • For mobile Apps and tracking purposes, use watchPosition() which keeps updating your lat/long as you move (use clearWatch() to stop tracking)
