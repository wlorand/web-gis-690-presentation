Web GIS 690 Presentation: D3 and D3 Projections

D3 Reference Map of the UK (D3.js, SVG, TopoJSON) - (JS for Web GIS - Spring 2016)

As part of the research for a geospatial web project, this implementation of D3 creator Mike Bostock's seminal "Let's Make a Map" tutorial proves that the influential JS data-visualization library can be used to make full-on cartographic reference maps. In addition to typical D3 data visualization operations such as appending svg <path> elements for the country boundaries and using geoJSON data values to render the major cities, this project also involved the use of D3's geo projection engine, and data format transformations using the Geospatial Data Abstraction Library's ogr2ogr tool on the command line.
